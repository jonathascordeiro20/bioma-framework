# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, attack arguments — never people —
and keep discussions technical and evidence-based.

Reports of unacceptable behavior: `jonathas.cordeiro2023@gmail.com`. All reports
are reviewed and kept confidential.
